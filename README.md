# Srijita Bandopadhyay Portfolio

This repository contains a static GitHub Pages portfolio adapted from the Google Sites portfolio at:

https://sites.google.com/view/srijitabandyopadhyay

## How to publish on GitHub Pages

1. Create or open the GitHub repository named `your-username.github.io`.
2. Add these files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Commit and push to the `main` branch.
4. In GitHub, go to **Settings -> Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root`, then save.

Your portfolio will be available at:

`https://your-username.github.io`

## Editing

- Update content in `index.html`.
- Update colors, spacing, and layout in `styles.css`.
- The navigation behavior is in `script.js`.

For a personal GitHub Pages site, the repository name should exactly match your GitHub username followed by `.github.io`.
